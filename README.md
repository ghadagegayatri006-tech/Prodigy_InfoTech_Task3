# Prodigy_InfoTech_Task3
Bank Marketing Prediction using Decision Tree
This project uses a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit based on demographic and campaign-related data.
# Objective
To build a machine learning model that classifies customer subscription (Yes/No) using the Bank Marketing dataset.
# Technologies Used
Python
Pandas
Matplotlib & Seaborn
Scikit-learn
# Model
Decision Tree Classifier
Criterion: Gini / Entropy
# Result
The model successfully predicts customer subscription behavior.
Call duration was observed as one of the most important features.

# Conclusion
This project demonstrates how a Decision Tree Classifier can be applied to real world marketing data to predict customer purchase behavior
